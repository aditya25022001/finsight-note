Note Taking App Server

This is the server of my mini project note taking app.

The following are the routes of the api

Method Route Parameters Access Description
POST /api/auth/register body : JSON{name, email, password} Public A route for user to register
POST /api/auth/login body : JSON{email, password} Public A route for user to login
GET /api/note/getnotes headers : Authorization : Bearer token Private A route for user to get all notes
PUT /api/note/updatenote
  • headers : Authorization : Bearer token
  • body : JSON{id, heading, content, tags}
 Private A route for user to update a particular note
DELETE /api/note/deletenote/id param : id Private A route for user to delete a note with id
POST /api/note/add
  • headers : Authorization : Bearer token
  • body : JSON{heading, content, tags}
 Private A route for user to add a note
